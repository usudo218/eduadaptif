Teknologi Informasi dan Komunikasi (Aplikasi Perkantoran)

Aperpersepsi

Sebagai pelajar SMA, Anda membutuhkan aplikasi perkantoran untuk menunjang aktivitas sekolah seperti:

  • Membuat laporan/tugas.
  • Membuat surat kegiatan acara.
  • Mengolah angka dan membuat presentasi.

Dalam penggunaannya, kita sering melakukan Integrasi Konten. Contohnya adalah menggunakan fitur Cut, Copy, Paste, menyisipkan gambar, hingga teknik Object Linking and Embedding (OLE). Selain itu, terdapat fitur lanjutan seperti Mail Merge, pembuatan daftar isi otomatis, dan pembuatan video presentasi.

Mengenal Aplikasi Perkantoran (Office)

Definisi: Aplikasi perkantoran adalah kumpulan aplikasi untuk memudahkan pekerjaan kantor. Karena berupa aplikasi siap pakai, pengguna tidak perlu membangun sistem dari awal dengan bahasa pemrograman.

3 Jenis Utama Aplikasi Perkantoran:

  1. Pengolah Kata (Word Processor):Untuk dokumen, surat, dan laporan.
    • Contoh:Microsoft Word, OpenOffice Word, Google Docs.
  2. Pengolah Lembar Kerja (Spreadsheet):Untuk mengolah data dalam tabel dua dimensi (kolom & baris) dan menampilkan diagram.
    • Contoh:Microsoft Excel, OpenOffice Spreadsheet, Google Sheets.
  3. Pembuat Presentasi (Slide):Untuk membuat slide yang menarik dengan butir penjelasan dan ilustrasi.
    • Contoh:Microsoft PowerPoint, OpenOffice Presentation, Google Slides.

Integrasi Konten Aplikasi Perkantoran

Apa itu Integrasi? Menurut KBBI, integrasi adalah penggabungan aktivitas atau komponen yang berbeda ke dalam satu unit fungsional. Dalam konteks perangkat lunak, integrasi bertujuan agar fitur-fitur dari berbagai aplikasi dapat digunakan bersama oleh pengguna.

Manfaat Integrasi:

  • Meminimalisir kesalahan manual (human error).
  • Menghindari duplikasi data.
  • Meningkatkan efektivitas komunikasi dan mempercepat sistem kerja.

Metode Integrasi:

  1. Perintah Cut, Copy, dan Paste:Menggunakan clipboard (buffer penyimpanan sementara).
  2. Fitur Integrasi di Aplikasi:Contohnya di MS Word, menu Insert Chart akan otomatis membuka MS Excel.
  3. Object Linking and Embedding (OLE):
    • Linking:Menghubungkan aplikasi. Jika data sumber berubah, data di aplikasi tujuan ikut berubah.
    • Embedding:Mentransfer data. Jika data di aplikasi tujuan diubah, data sumber asli tidak terpengaruh.

Tutorial Teknis (MS Office 2007/Versi Terbaru)

Object Linking (Excel ke Word)

  1. Siapkan file Excel (misal: xls).
  2. Buka MS Word, letakkan kursor.
  3. Klik tab Insert$\rightarrow$ grup Text $\rightarrow$ klik Object.
  4. Pilih Create from File$\rightarrow$ Browse $\rightarrow$ Pilih file Excel.
  5. Centang Link to file$\rightarrow$ OK.

Object Embedding (Paste Special)

  1. Blok data di Excel $\rightarrow$ Copy($Ctrl+C$).
  2. Buka MS Word $\rightarrow$ menu Home$\rightarrow$ Paste $\rightarrow$ Paste Special.
  3. Pilih Microsoft Office Excel Worksheet Object$\rightarrow$ OK.

Fitur Lanjut Aplikasi PerkantoranMail Merge

Digunakan untuk mengotomatiskan penulisan nama dan alamat surat massal.

  • Dokumen Master:Surat dengan bagian informasi yang dikosongkan.
  • Data Source:File (Excel) berisi daftar penerima.
  • Langkah:Tab Mailings $\rightarrow$ Select Recipients $\rightarrow$ Use Existing List $\rightarrow$ Insert Merge Field $\rightarrow$ Finish & Merge.

Membuat Daftar Isi Otomatis

  1. Terapkan Heading Stylespada setiap judul bab/sub-bab.
  2. Letakkan kursor di halaman daftar isi.
  3. Klik tab References$\rightarrow$ Table of Contents $\rightarrow$ Pilih gaya otomatis.

Membuat Video Presentasi (MS PowerPoint)

Fitur ini tersedia mulai versi 2013 ke atas.

  1. Buka PowerPoint, tampilkan tab Recording.
  2. Klik Record Slide Show. Anda bisa merekam suara dan wajah (via webcam).
  3. Gunakan Pen/Highlightuntuk menandai slide saat presentasi.
  4. Simpan hasil dengan cara: Export to Video$\rightarrow$ Create a Video (format .mp4).
