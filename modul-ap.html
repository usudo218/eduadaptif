1. Apa itu Algoritma?

Algoritma adalah urutan langkah-langkah logis untuk menyelesaikan sebuah masalah. Komputer hanya bisa bekerja jika diberikan instruksi yang jelas, urut, dan tidak bermakna ganda (ambigu).

2. Struktur Dasar Pemrograman

Pemahaman struktur dasar sangat penting sebelum mulai menulis kode program. Ada tiga struktur utama:

  • Sekuensial: Instruksi dijalankan baris demi baris dari atas ke bawah.
  • Percabangan (Branching): Program memilih jalur berdasarkan kondisi tertentu (Contoh: if-else).
  • Perulangan (Looping): Program mengulang instruksi yang sama berkali-kali (Contoh: for, while).

3. Variabel dan Tipe Data

Variabel adalah wadah untuk menyimpan data. Setiap data memiliki tipe, seperti Integer (angka bulat), String (teks), dan Boolean (logika).

4. Contoh Logika Percabangan

IF nilai > 75 THEN
  PRINT "Tuntas"
ELSE
  PRINT "Remedial"
